# HealthAI – AI-Powered Health Risk Prediction Platform

A full-stack web application that predicts health risks using machine learning and displays results through an interactive dashboard.

## Features
- Health risk prediction (ML-based simulation)
- User authentication (basic)
- Dashboard visualization

## Tech Stack
- Python (Flask)
- JavaScript
- Chart.js (optional)

## Run

cd backend  
pip install -r requirements.txt  
python app.py  

Open frontend files in browser
